## Overview

The on-prem collector runs within your network and processes Git repository data locally. Only metadata and analysis results are sent to the Navigara cloud.

## Architecture

The collector connects to your Git repositories, analyzes commits, and sends structured results to the Navigara API.

## Prerequisites

- Docker or Kubernetes runtime
- Network access to your Git repositories
- Outbound HTTPS to Navigara API
