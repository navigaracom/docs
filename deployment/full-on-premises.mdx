## Overview

Full on-premises deployment runs all Navigara components in your infrastructure, providing complete data sovereignty.

## Components

- **Backend** — Go gRPC/HTTP service
- **Frontend** — Next.js web application
- **PostgreSQL** — Primary data store
- **Collector** — Git repository analyzer
- **Task Processor** — Background commit analysis workers
