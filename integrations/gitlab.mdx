## Overview

Navigara supports both GitLab.com and self-hosted GitLab instances.

## Required Permissions

- Repository read access (code, commits, merge requests)

## Setup

1. Go to **Settings > Integrations**
2. Click **Connect GitLab**
3. Provide your GitLab instance URL (for self-hosted) and access token
4. Select the repositories you want to analyze
