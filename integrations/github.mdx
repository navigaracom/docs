## Overview

Navigara connects to GitHub via OAuth to analyze your repositories.

## Required Permissions

- Repository read access (code, commits, pull requests)

## Setup

1. Go to **Settings > Integrations**
2. Click **Connect GitHub**
3. Authorize the Navigara OAuth application
4. Select the repositories you want to analyze
